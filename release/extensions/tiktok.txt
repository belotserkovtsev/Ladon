# ladon extension: TikTok / ByteDance
#
# Подключается через config.yaml:
#   extensions: [tiktok]
#
# eTLD+1 домены TikTok и ByteDance-инфраструктуры (overseas edition).
# Все поддомены покрываются автоматически — ladon раскрывает manual-allow
# по eTLD+1 в dnsmasq ipset= директивах.
#
# Scope: только TikTok. Toutiao (новости) и defunct-продукты (musical.ly,
# Hypstar, Vigo) намеренно не включены — живую инфраструктуру TikTok они
# не затрагивают. Для полной картины ByteDance-сервисов добавьте свои
# домены в manual-allow.txt или заведите отдельный preset.
